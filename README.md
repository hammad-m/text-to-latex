# text-to-latex
This program is written in Java and is GUI based. It aims to make creating simple pdfs in latex easier. 


# It can parse following into latex code
1. Setting Page Size
2. Title
3. Subtitle
4. Author
5. Mail
6. Date
7. Section
8. Subsection
9. Text
10. Path (directory)
11. Code
12. List
13. Link

# Getting the pdf
Paste the code in any tex editor to parse it.
